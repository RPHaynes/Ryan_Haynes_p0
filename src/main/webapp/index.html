Contact Book

Ever wanted to have details of every person you know in one place? Well here is your chance with the new and completely innovated Contact Book!!!

Operations

    Get: gets entries based on parameters
    Post: adds entry based on JSON body
    Put: updates entries based on parameters and JSON BODY
    Del: deletes entries based on parameters

Operation Parameters

These Parameters can be used with operations (except for post) to specify what to search for

Parameters   Type  Example 
first_name  String  Ryan 
last_name  String  Haynes 
streetName  String  123 Normal BLV 
cityName  String  Indianapolis 
stateName  String  Indiana 
zipCode  int  47800 
phoneNumber  String  (812)251-0000 
birthDay  Local Date  1998-09-10 

JSON Entry Structure

{ "firstName": "Ryan", "lastName": "A", "streetName": "Street3", "cityName": "City1", "stateName": "NM", "zipCode": 540000, "phoneNumber": "1 (812) 251 0000", "birthDay": [ 1998, 9, 1 ] }

Get example

curl --location --request GET 'localhost:8080/api?birthDay=1998-09-10'

Post example

curl --location --request POST 'localhost:8080/api' \ --header 'Content-Type: application/json' \ --data-raw '{ "firstName": "Ryan", "lastName": "H", "streetName": "Street3", "cityName": "City1", "stateName": "Indiana", "zipCode": 540000, "phoneNumber": "1 (812) 251 0000", "birthDay": [ 1998, 9, 10 ] }'

Put example

curl --location --request PUT 'localhost:8080/api?lastName=A' \ --header 'Content-Type: application/json' \ --data-raw '{ "firstName": "Ryan", "lastName": "A", "streetName": "Street3", "cityName": "City1", "stateName": "NM", "zipCode": 540000, "phoneNumber": "1 (812) 251 0000", "birthDay": [ 1998, 9, 1 ] }'v

Del example

curl --location --request DELETE 'localhost:8080/api?firstName=Ryan'
